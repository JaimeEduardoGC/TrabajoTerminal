Sistema de información delictiva v.Alpha

Proyecto de tesis para el control del historial de faltas administrativas y/o delitos registrados en la Cárcel Municipal de Lázaro Cárdenas.

Código fuente

Funciones

  • Altas, bajas, cambios de usuarios y perfiles de infractores.
  • Altas, bajas, cambios de tipos de faltas administrativas y reportes.
  • Búsqueda de perfiles de infractores.
  • Manejo de sesiones.

Características

  • Diseñado sobre el framework CodeIgniter v2.1.3 cor arquitectura MVC que separa lógica, interfaz y acceso a base de datos propiciando la reutilización de código.
  • La interfaz se basa en Bootstrap de Twitter, permitiendo una presentación limpia, adaptativa y compatible con todos los navegadores actuales sin necesidad de código extenso.
  • Cada parte del sistema posee su propio módulo en application/modules, favoreciendo la escalabilidad e importación de código para aplicaciones similares.
  • Implementa seguridad nativa de CodeIgniter contra ataques XSS, CSRF e inyecciones SQL.
